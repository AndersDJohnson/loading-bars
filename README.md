loading-bars
============

Loading bars like Gmail app for Android.

## Install

Install with bower:

```sh
$ bower install --save loading-bars
```

## Use

Include `src/bars.js` on your page.

### Simple

```js
var bars = new Bars();
bars.start();

```

### Custom colors

```js
var bars = new Bars(['red', 'orange', 'yellow', 'green', 'blue', 'purple']);
bars.start();

```
